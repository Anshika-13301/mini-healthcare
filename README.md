
# Mini Healthcare Support Web App (AI-Enabled)

## Description
This project is a full-stack AI-enabled healthcare support web application. NGOs can collect patient and volunteer requests efficiently, with automated responses for faster handling.

## Tech Stack
- Frontend: React.js (used css)
- Backend: Node.js, Express

## Features
- Patient/volunteer request form submission
- Automated AI-like response
- Data stored in MongoDB
- Separate frontend & backend

## Folder Structure

mini-healthcare/
├── backend/ # Node.js + Express server
├── frontend/ # React frontend
├── README.md

## Setup Instructions

### Backend
1. Navigate to backend:
```bash
cd backend
node server.js
```
